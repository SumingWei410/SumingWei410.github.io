取整函数

前言

在高中的学习中，我们常常研究一些连续性的函数，而忽视了对整数的研究。事实上，整数在数学里也是不可忽视的一部分，毕竟我们最开始接触数学时也是整数，而且整数在高等数学里也是十分重要的研究对象。下面就一起来看看与整数密不可分的函数——取整函数吧

化小数为整数

假设有一段绳子长5.2m，现在需要几段1m的绳子做标准材料，请问最多能截出多少根标准材料？答案显而易见，是5根。但如果把绳子的长度换成xxxm呢？此时若需要一个数学符号来表示这结果（含xxx），就需要取整函数[]\left [ \right ][]出马了。
定义：用[x]\left [ x \right ][x]表示不超过xxx的最大整数，即x−1<[x]≤xx-1 < \left [ x \right ] \le xx−1<[x]≤x
例如[3.4]=3\left [ 3.4 \right ] = 3[3.4]=3，[0.2]=0\left [ 0.2 \right ] = 0[0.2]=0，特别注意，[x]\left [ x \right ][x]是不超过xxx的最大整数，所以负数应写成这样：[−2.3]=−3\left [ -2.3 \right ] = -3[−2.3]=−3
这样，我们就完成了小数到整数的转换
